# py_tictactoe

Tic-tac-toe game implemented in Python, played in console.

## Features

- This game is played in console. The tic-tac-toe grid is re-printed as the game goes on.
- Players choose the spot by number to mark their shape.
- Player X goes first.
- When a player reaches three in a row, the game ends announcing the winner.
- If the players tie, the game ends in a draw.

## Implementation

The game loop contains functions that handle each step of the game independently.
